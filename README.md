# Telegram to Obsidian Converter

Конвертирует экспорт Telegram в формат Obsidian Vault с поддержкой медиа-файлов, форматирования текста и полнотекстовой индексации.

## Возможности

✨ **Форматирование текста**
- Жирный, курсив, код, цитаты
- Ссылки, упоминания, спойлеры
- Подчёркивание и зачёркивание

📎 **Медиа-файлы**
- Фото, видео, аудио, голосовые сообщения
- Стикеры и документы
- Автоматическая дедупликация по хешу

🔍 **Поиск и индексация**
- JSONL индекс для быстрого поиска
- JSON выборка первых 1000 сообщений
- Полнотекстовый поиск в Obsidian

📅 **Организация**
- Группировка по датам и чатам
- Ссылки между сообщениями (reply_to)
- Автоматическое создание структуры папок

## Требования

- Python 3.8+
- ijson (для обработки больших JSON файлов)
